# 公开仓库隐私检查

本仓库公开发布前必须满足：

1. 公开文字仅使用“本机 Telegram 桥接服务”等通用称呼，不包含具体机器人用户名。
2. 不包含 Telegram Session、手机号、账号、会话数据库或桥接端私有配置。
3. 不包含 115 Cookie、Token、API Key、密码或真实分享链接。
4. 不包含用户个人域名、NAS 内网 IP、容器日志或数据库。
5. 桥接服务 URL 和分类服务 URL 由 MoviePilot 运行时配置保存，源码默认值不得写入用户专属地址。
6. PT 入库屏障只能读取 MoviePilot/媒体库正式缺失状态，不得把个人媒体路径、媒体库 ID 或运行日志写入仓库。
